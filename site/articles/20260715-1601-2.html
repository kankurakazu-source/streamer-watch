ロジクールGの新ゲーミングデバイス2機種が16日発売、G316 X 98とG304 X SUPERLIGHTを比較する

2026/07/15 ・ 文: ガジェゲ編集部
※本記事にはアフィリエイト広告（プロモーション）が含まれます。

ロジクールGブランドから、ドットマトリックスディスプレイ搭載キーボード「G316 X 98」と、40g軽量化を実現したマウス「G304 X SUPERLIGHT」が7月16日に発売される。両機種とも既存モデルからの明確な進化点があり、周辺機器の買い替えを検討している層には見逃せない発売タイミングだ。

結論
軽量マウスを求めるなら「G304 X SUPERLIGHT」、情報表示や打鍵体験を重視するなら「G316 X 98」が狙い目だ。
目次
  1. 「G316 X 98」はドットマトリックスディスプレイが最大の特徴
  2. 「G304 X SUPERLIGHT」は従来比40g軽量化を実現
  3. 同日発売の意味と選び方

「G316 X 98」はドットマトリックスディスプレイが最大の特徴

ロジクールGの新型ゲーミングキーボード「G316 X 98」は、98％レイアウトにドットマトリックスディスプレイを搭載した点が特徴だ。ディスプレイ搭載キーボードはこれまで上位モデルに限られていた機能で、視覚的な情報表示をコンパクトなレイアウトに落とし込んだ点は注目に値する。発売は7月16日で、価格は現時点で公式発表待ちの状態だ。

「G304 X SUPERLIGHT」は従来比40g軽量化を実現

マウス側の新型「G304 X SUPERLIGHT LIGHTSPEED ワイヤレス ゲーミング マウス」は、従来モデルから約40gの軽量化を果たした点が最大の売りだ。軽量ワイヤレスマウスはFPSやMOBAといった俊敏な操作を要するタイトルで恩恵が大きく、長時間プレイでの疲労軽減にも直結する。こちらも発売は7月16日で、キーボードと同日展開となる。

同日発売の意味と選び方

キーボードとマウスが同日に発売されるのは、周辺機器を一式で刷新したいユーザーにとって都合が良いタイミングといえる。ディスプレイ機能を活かした情報管理を重視するなら「G316 X 98」、操作の軽さと反応速度を最優先するFPS・MOBA系プレイヤーなら「G304 X SUPERLIGHT」が候補になる。両機種ともロジクールGブランドの現行ラインで、既存モデルからの明確な差別化点を持つのが強みだ。

まとめ

2機種とも7月16日発売で、性格の異なる進化を遂げている点が興味深い。購入を検討する場合は、ディスプレイ機能の実用性と軽量化のメリットのどちらを優先するかで選び分けるのが良い。
